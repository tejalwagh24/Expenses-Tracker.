# Expenses-Tracker.
Python project for tracking daily expenses.
## Features
- Add new expenses
- View all expenses
- View total and category-wise expenses
- Save data in a CSV file

##  Technologies Used
- Python
- CSV Module
- Datetime Module
- File Handling

1. Download the project files.
2. Open terminal and run:
